About Englandball Editor

イギリスボール公認エディタは、Web標準の先端技術を集約したエクスプレッシブな開発用PWAカタログです。

  • M3 (Material Design 3): Google Sans、精緻なコンポーネント、調和されたマテリアルカラー
  • ディレクトリ同期: HTML5 File System Access API に完全対応
  • プレビュー機構: 複数ファイルが相互作用する静的サイトもBlob上で結合・仮想起動
  • Git統合: isomorphic-git によるWebクライアント上でのクローンおよびファイルシステム格納
